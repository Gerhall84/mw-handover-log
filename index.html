One-time setup. Paste your Supabase project URL and anon public key (Supabase → Project Settings → API). These get saved in this browser; once you've hosted the file, bake them into the config block at the top of the script so every user is connected automatically.

The anon key is the public key — safe in client apps. Access is controlled by the policies in the setup SQL.
